# General-Domain Arabic Glossary

High-frequency Arabic terms that recur in non-legal translation work (business correspondence, news, technical documentation, marketing). Use these consistently when running in **general mode**. For legal-domain terms, see `glossary-legal.md`.

## Numbers and Quantities

| Arabic | English | Chinese |
|---|---|---|
| رقم | number | 编号 / 号码 |
| عدد | count / quantity | 数量 |
| مئة / مائة | hundred | 百 |
| ألف | thousand | 千 |
| مليون | million | 百万 |
| مليار | billion | 十亿 |
| نسبة | percentage / ratio | 百分比 / 比率 |
| نسبة مئوية | percent | 百分之…… |
| زيادة | increase | 增加 |
| نقصان | decrease | 减少 |
| إجمالي | total | 总计 |
| متوسط | average | 平均 |

Note on Arabic-Indic digits: ٠١٢٣٤٥٦٧٨٩ (Eastern Arabic numerals) are common in Saudi text. Western digits 0–9 may be used in EN/ZH output for readability; flag this choice if the number is a reference identifier.
